# ⚔️ FeliciaMaldosa

Em uma grande cidade dominada pelo caos emocional, egoísmo, falsidade e relações tóxicas, surge **FeliciaMaldosa**.

Diferente das histórias comuns, seus inimigos não são monstros.

Seus verdadeiros inimigos são:
- pessoas narcisistas;
- manipuladores;
- relações destrutivas;
- conflitos internos;
- ansiedade;
- medo;
- tristeza;
- e os desafios diários da vida moderna.

A cada batalha enfrentada no cotidiano, Felicia ganha XP.

Mas esse XP não representa apenas experiência.

Representa:
- força emocional;
- sabedoria;
- inteligência social;
- resistência mental;
- autocontrole;
- evolução pessoal.

Enquanto muitos desistem diante das dificuldades, Felicia enfrenta duas guerras ao mesmo tempo:
- a batalha externa contra o mundo;
- e a batalha interna contra suas próprias emoções.

E a cada dia vencido, ela se torna mais forte.

---

# 🧠 Sistema de Níveis

Se XP for menor do que 1.000 = Ferro  
Se XP for entre 1.001 e 2.000 = Bronze  
Se XP for entre 2.001 e 5.000 = Prata  
Se XP for entre 5.001 e 7.000 = Ouro  
Se XP for entre 7.001 e 8.000 = Platina  
Se XP for entre 8.001 e 9.000 = Ascendente  
Se XP for entre 9.001 e 10.000 = Imortal  
Se XP for maior ou igual a 10.001 = Radiante  

---

# 💭 Filosofia de FeliciaMaldosa

> “A verdadeira força não nasce quando tudo está bem.
> Ela nasce quando você continua avançando mesmo quebrada por dentro.”

# 💻 Objetivo do Projeto

Este projeto representa evolução.

Não apenas evolução em programação, mas também:
- evolução mental;
- crescimento pessoal;
- superação diária;
- disciplina;
- resistência emocional.

Cada linha de código representa uma batalha vencida.

Cada erro corrigido representa XP adquirido.

Cada desafio superado aproxima FeliciaMaldosa do nível mais alto:

# 👑 RADIANTE
